# Super simple Neural Networks library

This library is just a learning project based on <https://github.com/CodingTrain/Toy-Neural-Network-JS> 
and <http://neuralnetworksanddeeplearning.com/>

### Development

#### Compiling

To compile you can do:

```bash
cmake -B build .
cd build
make
```

and to test it:

```bash
cd build
./tests
```
